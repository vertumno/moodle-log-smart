# Workflow: Greenfield UI Development

**Type:** Workflow
**Status:** Active
**Version:** 1.0.0
**When to Use:** For developing a new frontend/UI application from scratch

---

## Overview

Cria e desenvolve uma aplicação frontend nova a partir do zero.

## Fases

1. Setup de projeto (framework, dependências)
2. Configuração de base layout e routing
3. Criação de componentes principais
4. Implementação de páginas
5. Integração com API
6. Testes
7. Documentação
8. Deployment

---

**Para iniciar:** `*workflow greenfield-ui`
