# Workflow: Greenfield Full Stack Development

**Type:** Workflow
**Status:** Active
**Version:** 1.0.0
**When to Use:** For developing complete features in a new full-stack project

---

## Overview

Orquestra desenvolvimento em um projeto novo (greenfield), do zero.

## Fases

### 1️⃣ Setup Inicial

- [ ] Criar branch principal
- [ ] Configurar estrutura de pastas
- [ ] Instalar dependências
- [ ] Configurar environment files
- [ ] Inicializar testes

### 2️⃣ Desenvolvimento Backend

- [ ] Criar API structure
- [ ] Implementar endpoints
- [ ] Configurar banco de dados
- [ ] Implementar autenticação
- [ ] Adicionar testes

### 3️⃣ Desenvolvimento Frontend

- [ ] Configurar framework UI
- [ ] Criar componentes base
- [ ] Implementar páginas
- [ ] Integrar com API
- [ ] Adicionar testes

### 4️⃣ Testes Integrados

- [ ] Testes end-to-end
- [ ] Performance testing
- [ ] Security validation

### 5️⃣ Deploy & Documentation

- [ ] Criar documentação
- [ ] Configurar CI/CD
- [ ] Deploy para staging/production

---

**Para iniciar:** `*workflow greenfield-fullstack`
