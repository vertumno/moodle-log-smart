# Workflow: Greenfield Service Development

**Type:** Workflow
**Status:** Active
**Version:** 1.0.0
**When to Use:** For developing a new backend service from scratch

---

## Overview

Cria e desenvolve um novo serviço backend a partir do zero.

## Fases

1. Planejamento e design de arquitetura
2. Setup de projeto (framework, dependências)
3. Implementação de handlers/controllers
4. Implementação de lógica de negócio
5. Testes completos
6. Documentação de API
7. Deployment

---

**Para iniciar:** `*workflow greenfield-service`
